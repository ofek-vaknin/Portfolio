Checkers Game

I developed a Checkers game featuring a Windows Forms graphical user interface as part of an Object-Oriented Programming (OOP) course project.

The application was designed and implemented as a desktop Windows-based system to visualize and manage gameplay, applying core OOP concepts.

My contributions included implementing the game logic, turn-based mechanics, and interactive UI components.

Technologies used: C#, .NET Framework, Windows Forms.
