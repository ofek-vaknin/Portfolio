Task Management System

Java-based desktop application for managing tasks with structured architecture and design patterns.

Java Swing Design Patterns DerbyDB
View on GitHub Back to Projects

Overview

Task Management System is a Java-based desktop application developed as part of an advanced software engineering course.

The system enables users to create, update, and manage tasks while demonstrating clean architecture and separation of concerns.

The project emphasizes real-world object-oriented design and extensive usage of design patterns.

Technologies

Java Swing DerbyDB MVC / MVVM OOP

Key Features

  • Task creation, editing, and status management
  • Clean layered architecture with separation of concerns
  • Use of mandatory design patterns (Combinator, Visitor)
  • Additional patterns: Singleton, Proxy, Observer, State
  • Persistent storage using DerbyDB (DAO pattern)
