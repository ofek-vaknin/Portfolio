Overview

InterviewLog is a full-stack web application designed to help developers track and manage job interviews.

The system includes authentication, interview status management, and a visual dashboard presenting interview statistics.

The project was built with a strong focus on clean UI, scalable architecture, and real-world workflows.

Technologies

React Firebase Auth Firestore Material UI JavaScript

Key Features

  • User authentication (Register / Login / Reset password)
  • Interview status tracking (Kanban-style workflow)
  • Dashboard analytics & statistics overview
  • Real-time data sync with Firebase Firestore
  • Responsive UI with Material UI components
